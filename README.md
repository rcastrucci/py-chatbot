# PY-CHATBOT
###### Still in development

#### Working on a NLP chatbot not using any libraries

## TO RUN
copy commands bellow, paste into your terminal and press enter

## **MacOS**
    git clone https://github.com/rcastrucci/py-chatbot.git
    cd py-chatbot
    ./py-chatbot

## **Windows or Linux**
    git clone https://github.com/rcastrucci/py-chatbot.git
    cd py-chatbot
    python3 py-chatbot
